# Nuxt 3 Minimal Starter custom for yue

Look at the [nuxt 3 documentation](https://v3.nuxtjs.org) to learn more.

> unocss + nuxt3

## Setup

Make sure to install the dependencies:

```bash
# pnpm
pnpm i
```

## Development Server

Start the development server on http://localhost:8989

```bash
pnpm run dev
# or
pnpm run start
```
